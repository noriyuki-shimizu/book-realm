# Book Realm

書籍管理アプリです。
個人の書籍を管理できます。
画像を解析させ、登録を補助します。
その他、フォームからの入力も行えます。

## アプリケーションリンク

https://book-realm-xi.vercel.app/

## Develop Guid

### Setup

Make sure to install the dependencies:

```bash
npm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

### Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

### Linter

```bash
# ESLint / dprint / StyleLint
npm run lint
```

### Format

```bash
# ESLint / dprint / StyleLint
npm run format
```

### Unit Test

```bash
# Vitest
npm run test:ut
```
